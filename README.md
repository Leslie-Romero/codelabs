## Puntos de interés

En su mayoría, con el tiempo limitado por la carga de trabajo de estas últimas semanas, he podído solo hacer algunos de los codelabs más básicos, para familiarizarme con el sistema, es decir, como se componen las funciones "Composables", la sintaxis del lenguaje Kotlin, el rol de los fragmentos, las "activities" y los modelos, como modificar las funciones Composables usando "row", "surface", etc. También he querido centrarme en el diseño de la interfaz de usuario, ya que es una parte que me resulta muy interesante por lo que he revisado la sección de Theming, pero me ha sido difícil encontrar tiempo para completarlo. Al final he podido solo trabajar ligeramente con la conexión de la base de datos para poder adaptarlo a nuestro proyecto final, sobre todo como conectar el proyecto de Firebase con nuestro proyecto en Android Studio, las dependencias y plugins necesarios para que todo funcione correctamente, y el uso del simulador de Firebase para aprender a utilizar las funcionalidades de Firestore.
